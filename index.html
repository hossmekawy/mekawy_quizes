Question 1

What is a key advantage of RTOS?

Correct Answer: C. Maximum utilization of devices

RTOS provides maximum utilization of devices and resources through efficient scheduling and real-time capabilities.

Question 2

What is the primary purpose of an Operating System (OS)?

Correct Answer: D. To manage hardware and software resources

The primary purpose of an OS is to manage and coordinate hardware and software resources efficiently.

Question 3

What is a Real-Time Operating System (RTOS)?

Correct Answer: D. An OS that responds within a defined time limit

RTOS is designed to respond to events within a predictable and defined time constraint.

Question 4

What is the main function of the dispatcher in an operating system?

Correct Answer: C. To give CPU control to the selected process

The dispatcher's main function is to give CPU control to the process selected by the scheduler.

Question 5

What is the primary use case for Soft RTOS?

Correct Answer: A. Systems where occasional delays are tolerable

Soft RTOS is used in systems where missing deadlines occasionally is acceptable and doesn't cause system failure.

Question 6

In an operating system, what is the kernel responsible for?

Correct Answer: A. Managing hardware and system resources

The kernel is the core component responsible for managing hardware and system resources.

Question 7

Multitasking system where multiple users share CPU.

Correct Answer: C. Time-Sharing OS

Time-sharing OS allows multiple users to share CPU time through multitasking.

Question 8

What is the key advantage of a distributed operating system?

Correct Answer: A. It allows multiple computers to function as a single system

Distributed OS enables multiple computers to work together as a unified system.

Question 9

Which RTOS task property specifies the memory reserved for its execution context?

Correct Answer: A. Stack size

Stack size specifies the memory reserved for a task's execution context.

Question 10

Which of the following best describes a program?

Correct Answer: D. A passive entity stored on disk

A program is a passive entity stored on disk, while a process is the executing instance.
